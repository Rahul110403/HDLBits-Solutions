# HDLBits - 7458

## Problem Summary

Implement the logic of the 7458 integrated circuit using Verilog HDL.

The circuit generates two output signals:
- **p1y** from two 3-input AND gates followed by an OR gate.
- **p2y** from two 2-input AND gates followed by an OR gate.

## Concepts

- Continuous Assignments
- Bitwise AND (`&`)
- Bitwise OR (`|`)
